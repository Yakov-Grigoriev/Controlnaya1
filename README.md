# Игра "Угадай число"
«Угадай число» —
популярная логическая игра, цель которой — отгадать загаданное компьютером или другим игроком число за минимальное количество попыток. Обычно загадывается целое число в определенном диапазоне (например, от 1 до 100). После каждого варианта игрока программа подсказывает, загаданное число больше или меньше введенного, помогая сузить диапазон поиска. 
Основные характеристики и правила игры:

    Механика: Игрок вводит число, компьютер сравнивает его с загаданным.
    Подсказки: Программа сообщает: «Больше», «Меньше» или «Угадано».
